About the application

This application is a simple web application that allows you to predict your country's emissions from Agricultural sources, using simple inputs such as the country's GDP, population, and percentage population in urban areas.

We hope that this application will help you understand the impact of urbanization on emissions from Agricultural sources, and how this can be used to inform policy decisions.
